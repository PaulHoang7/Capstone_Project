## Rule
Any bug or issue that takes more than 30 minutes to debug **must be logged here** before moving on.
Review this file before starting a new phase or experiment.

---

## Entry Template

When adding a new entry, use this format:

```
### [Category] Short description
- **Issue**: What happened?
- **Root cause**: Why did it happen?
- **Fix**: What was done to resolve it?
- **Lesson**: What to remember / avoid next time?
- **Date**: YYYY-MM-DD
```

### Categories
- `Dataset` — data, metadata, audio, text, tone extraction issues
- `Training` — training, config, checkpoint, OOM, convergence issues
- `Model` — architecture bugs, Dual-Path Encoder issues, cross-model issues
- `Evaluation` — metrics, test set, comparison errors
- `Deployment` — ONNX export, browser, quantization issues
- `Environment` — dependencies, GPU, storage, NFS, MFA setup
- `General` — anything else
